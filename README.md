# Movie Rating Prediction with Python

## Objective
Predict movie ratings using machine learning based on features such as Genre, Director, Actors, Duration and Votes.

## Dataset
IMDb Movies India Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Machine Learning Model
Random Forest Regressor

## Workflow
1. Data Collection
2. Data Cleaning
3. Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation

## Results

MAE: 0.83

R² Score: 0.35
